** 미니 특강 : WSL과 리눅스 **
(0) Windows Subsystem for Linux : 윈도10. 1주년기념에 출시.
  배경 : Windows 개발자 --> 맥(Mac)의 셸(다양한 명령어)
	명령 프롬프트 >> 파워셸 >> 리눅스를 포함
  버전 : 1.0(WSL1) >> 2.0(WSL2) >> 2.5.7.0

(1) WSL 기반 설치 (닌텐도 기계)
 wsl  --install  --no-distribution   ==> 설치 후 재부팅 (업데이트)
(2) 리눅스 다운로드 설치 (기본 : Ubuntu)
 wsl --install  -d  ubuntu   ==> 리눅스 설치
(3) 리눅스 사용자/비번 생성
(4) $ 프롬프트 ==> 완전한 리눅스 설치

# 추가 명령
wsl --list  --online  ==> 설치 가능한 리눅스 목록
wsl --list ==> 설치된 리눅스 목록
wsl --install  -d  배포이름
wsl --version ==>  설치된 WSL 버전 확인

## 미니 특강 : 리눅스 기본 명령어 ##
pwd : Print Working Direcoty. 현재 디렉터리 출력.
	C:/users/5-29   ==> 현 사용자의 홈 디렉터리
	/home/karina
cd 폴더 : Change Directory. 디렉터리 이동
cd : 현 사용자의 홈 디렉터리로 이동
cd .. : 바로 앞 폴더로 이동
ls : List. 폴더의 목록 보기
ls -l : 자세히 보기. d/-/l
ls -al : 숨김도 보기. 리눅스는 제일 앞에 (.) 있으면 숨김.
clear : 화면 깨끗히
mkdir : 디렉터리 생성
rmdir : 디렉터리 삭제
rm  -rf  폴더 : 폴더 삭제
touch 파일명 : 빈 파일 생성
rm -f 파일 : 파일 삭제
cp 원본 타겟 : Copy
mv  원본  타겟 : 이름 바꾸기
mv  파일  디렉터리 : 파일을 디렉터리로 이동
vi  파일명 > 명령모드 > i > 편집모드 >(작업) > Esc
	> 명령모드 >  :wq
cat  파일명
